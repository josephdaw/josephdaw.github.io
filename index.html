InstantCrypto more_vert

HTML | CSS | JavaScript | jQuery | Bulma | Font Awesome

GitHub Live page
InstantCrypto close

A cryptocurrency search tool for the latest price and news data.

This was a group project made up of Stefanie Buntz (https://github.com/sbuntz), Michael West (https://github.com/mjhwest), Gurtej Thandi (https://github.com/gurtej154), and myself.

We paired cryptocurrency data with recent and relevant news articles related to a specific cryptocurrency. This would allow a user to have a more streamlined experience and stay in one place, rather than checking the value of their currency and then having to open a new application to find out why the value has changed or increased.

NOTE: the subscription for the News API expires on 31 Oct 2021. Unfortunately, searches after the aforementioned date will not return news data.

Weather Dashboard more_vert

HTML | CSS | JavaScript | jQuery | Bulma

GitHub Live page
Weather Dashboard close

A weather dashboard that will run in the browser and feature dynamically updated HTML and CSS utilising the OpenWeather One Call API.

This project requires a good understanding of server-side APIs. I made use of an alternative framework to Bootstrap for this project. The framework I chose was Bulma. It has reasonably good documentation and has been relatively easy to implement.

I have tried to include as much functionality as I could. This lead me to the decision to allow the user to select the units they would like the weather to be displayed in. Throughout this project I have tried to minimise any repeated or unnecessary code as much as possible. At times, this streched my abilities and resulted in me breaking something, sometimes to the point I would have to backtrack to the previous Git Commit and start again. While this proved challenging and indeed frustrating, I believe that I have learned a lot from the process.
